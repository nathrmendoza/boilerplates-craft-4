# CRAFT 4 Boilerplate
A craft 4 build from scratch boilerplate that utilizes the following frontend tooling:
- webpack
- sass
- babel
- jquery
- slick carousel

## How to set up
- Extract files and folders into main directoy, overwrite all
- import db file at `/db/craft-db.sql.zip`
- run npm install
